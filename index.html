Frequently Asked Questions

Does WiFi Server Pro work over the internet?
No, it only works on your local WiFi network for privacy and security.
What permissions does the app require?
Storage, network, and notification permissions are required for file sharing and server status.
Is my data safe?
Yes. All file transfers stay on your local network. No data is collected or shared.
Can I change the shared folder?
Yes, you can customize the shared folder in the app settings.
Is there a file size limit?
No, the only limit is your device storage and browser/client capabilities.

Privacy Policy

WiFi Server Pro does not collect, store, or share any personal data.

Data Collection

  • The app does not collect any personal information.
  • Files are only accessible on your local network.
  • No analytics or tracking is used.

Permissions

  • Storage: Required to access and share files on your device.
  • Network: Required for server functionality on your network.
  • Notifications: Used to show server status and alerts.

Security

  • Supports HTTPS and FTPS for secure file transfers.
  • Authentication is available for both FTP and HTTP access.
  • Users should set strong passwords and manage access responsibly.
